# felixlynch.com

Terminal-style portfolio site showcasing my GitHub projects.

## Features

- Fake terminal UI with interactive commands
- Auto-fetches projects from GitHub API
- Dark theme with cyan accents (Hollow Knight inspired)
- Commands: `ls`, `cat <project>`, `help`, `about`

## Tech

- Vanilla HTML/CSS/JS
- GitHub REST API
- No build step, pure static files

## Development

```bash
# Serve locally
python3 -m http.server 8000
# Then visit http://localhost:8000
```

## Deploy

Static files, deploy anywhere (GitHub Pages, Netlify, etc.)
